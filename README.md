# worktime-calculator
# ⏱️ 工时看板 (Worktime Dashboard)

一个基于 Apple 设计规范的极简、高效的工时统计工具。它能帮助你精准掌控本月加班进度，自动计算剩余工作日的平均工时负担，并支持深色模式。

## ✨ 核心功能

* **🚀 智能进度追踪**：实时展示本月累计加班时长。
* **📊 动态负担计算**：自动根据本月剩余工作日（包含今日）计算每天需要加班的时长。
* **🎨 苹果视觉风格**：采用 **San Francisco** 数字字体与 **苹方** 中文字体，极简 UI 交互。
* **🌓 自动深色模式**：无缝适配系统主题，夜间加班不刺眼。
* **📍 本地数据存储**：所有工时记录保存于浏览器本地（LocalStorage），不上传服务器，保护隐私。
* **⚖️ 灵活校准**：支持手动编辑累计加班时长，方便与 HR 系统数据进行快速对齐。

## 🚦 状态识别逻辑

系统通过三种颜色直观反映你的加班达标状态（以 30H 为目标）：

* 🔴 **红色 (-5H 以上)**：距离目标缺口较大，需加油。
* 🟡 **黄色 (5H 以内)**：接近目标，进入冲刺阶段。
* 🟢 **绿色 (已达成)**：已满足目标，展示溢出时长。

## 🛠️ 技术实现

* **前端**: HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6)
* **字体族**: `-apple-system`, `PingFang SC`, `SF Pro Text`
* **存储**: 浏览器 `localStorage`
* **计算逻辑**:
* 正班工时上限设定为 **7.5H/天**。
* 加班起始时间锚定为 **18:30**。
* 工作日计算逻辑：自动排除周六日，包含今日在内进行分摊。

## 🔒 隐私声明

**本工具为纯前端应用。**

* 没有任何后端 API。
* 没有任何数据库。
* 没有任何埋点或追踪代码。
你的所有个人工时数据**永远不会离开你的设备**。
