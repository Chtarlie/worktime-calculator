# 📅 工时助手 (WorkHour Helper) V4.1

![Version](https://img.shields.io/badge/Version-4.1-blue.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

工时助手是一款基于纯前端技术构建的个人加班时长管理工具。它通过直观的环形进度条和动态压力算法，帮助用户更好地规划月度工时，拒绝盲目加班。



## ✨ 核心亮点

- **🎯 目标对冲算法**: 剩余需求不再是枯燥的剩余数，而是通过负数显示“当前缺口”，直观展示还需要努力多久。
- **📈 环形可视化**: 引入 SVG 动态渲染技术，让数据具备温度，颜色随进度变化。
- **⚡ 实时打卡**: 针对移动端优化，一键捕获当前时间，告别手动调整时间的 Bug。
- **🧠 智能分母逻辑**: 自动识别当日打卡状态。若已记录，日均压力计算自动剔除今日，确保预测的准确性。
- **🛡️ 隐私至上**: 无后端、无数据库、无 Cookie，所有数据本地化存储在您的浏览器中。

## 🛠️ 技术栈

- **HTML5**: 语义化结构及数据呈现。
- **CSS3**: 使用 Flexbox 与 Grid 混合布局，实现精密的首末对齐交互。
- **JavaScript**: 原生 Vanilla JS，负责高精度的时间换算与工作日逻辑算法。

## 🚀 快速启动

1. **保存文件**: 将代码另存为 `加班助手.html`。
2. **运行**: 双击文件通过浏览器打开。
3. **设置目标**: 在首页修改“目标”数值（例如 30H）。
4. **开始记录**: 下班时点击“打卡”，然后点击“保存今日记录”。
